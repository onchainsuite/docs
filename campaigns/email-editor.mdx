## Editing Checklist

- Write a subject line that matches the user intent.
- Add preview text that supports the primary action.
- Keep body copy easy to scan.
- Use one main CTA before adding secondary options.
- Double-check personalization and fallback values.

## Recommended Pattern

Lead with the user-relevant event, explain why it matters now, and give one next action.
