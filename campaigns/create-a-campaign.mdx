## When To Use This

Use a campaign when you want to send a one-time message such as a product update, mint reminder, rewards announcement, or ecosystem activation email.

## Steps

1. Open the target project and go to the campaign area.
2. Click `New Campaign` and choose an email campaign type.
3. Name the campaign clearly so reporting stays readable.
4. Select or create the audience segment you want to reach.
5. Choose a template or start from a blank editor.
6. Add subject line, preview text, and sender details.
7. Save a draft and hand it to teammates for review.

## Recommended Setup Checklist

- Use a campaign name that includes audience, goal, and date.
- Confirm the sender identity matches the project brand.
- Keep one primary CTA per email.
- Review links, personalization tokens, and fallback text.
- Validate the audience size before moving to testing.
