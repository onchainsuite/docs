# Onchain Suite Branding Guide

Source: designer-provided brand board screenshot shared on 2026-06-15.

This file is intended to be reusable in frontend codebases as a single source of truth for color, typography, gradients, and motif usage.

## Brand Positioning

Onchain Suite should feel:

- Crypto-native
- Trustworthy
- Technical, but not cold
- Clear for marketers and operators
- Premium without feeling corporate or over-designed

## Core Colors

### Primary Palette

- `Cloud`: `#F0F7FF`
- `Electric Blue`: `#1727E0`
- `Midnight`: `#010F31`
- `Sky`: `#2F94FF`
- `Accent Orange`: `#FF6828`

### Neutral Palette

- `White`: `#FFFFFF`
- `Black`: `#000000`

## Recommended Usage

- Use `#1727E0` as the primary brand color for actions, active states, and key highlights.
- Use `#010F31` for dark surfaces, strong text, and premium contrast moments.
- Use `#2F94FF` for secondary highlights, supporting UI emphasis, and lighter gradient transitions.
- Use `#F0F7FF` for soft backgrounds, cards, panels, and light UI framing.
- Use `#FF6828` sparingly for accents, alerts, and CTA emphasis.

## Typography

### Font Family

- Primary typeface: `Instrument Sans`
- Fallback stack: `Instrument Sans, Inter, ui-sans-serif, system-ui, sans-serif`

### Type Tokens

- `Heading`: `69px`, `700`, `-1px`
- `Subheading`: `48px`, `600`, `-1px`
- `Body`: `28px`, `400`, `0px`

These sizes come from the designer guide and appear to be reference sizes for brand comps rather than direct production UI sizes. For product UI, scale them into a practical type ramp while preserving the weight and spacing relationships.

## Brand Motif

The brand motif is derived from the Onchain Suite logo mark:

- Circular split form
- Vertical spine/stem
- Soft geometric edges
- Strong silhouette even in monochrome

### Recommended Motif Usage

- Faint background watermark
- Section divider or hero embellishment
- Mask or framing device for gradients
- Container shape for icons, avatars, or KPI art

### Avoid

- Stretching or distorting the motif
- Rotating it arbitrarily
- Using the motif at high opacity behind dense body text
- Combining too many gradients and motif layers in one component

## Implementation Notes

- Use `Midnight` and `White` for the strongest contrast pair.
- Use `Cloud` instead of pure white backgrounds when a softer product feel is needed.
- Reserve `Accent Orange` for emphasis rather than broad UI coverage.
- Use gradients on hero areas, banners, and charts more than on dense form-heavy screens.
- Preserve the logo and motif geometry consistently across docs, product, and marketing surfaces.
