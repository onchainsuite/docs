## Core Terms

### Workspace

The top-level container for your team, projects, and shared access.

### Project

The operational space where settings, audiences, campaigns, automations, and integrations live.

### Wallet-First Identity

The model where the wallet is the canonical user record and other channels attach when available.

### Audience

A set of users or wallets selected for messaging, analysis, or lifecycle action.

### Segment

A reusable audience definition created from rules, queries, or saved logic.

### Campaign

A one-time or scheduled outbound message to a defined audience.

### Automation

A triggered workflow that listens for an event, checks conditions, and sends or branches actions.

### Play

A pre-built automation pattern designed for a common Web3 lifecycle moment.

### Publishable Key

A client-safe key used for flows such as wallet registration or in-app challenge and verification.

### Secret Key

A server-only key used for secure protocol-to-platform operations.
