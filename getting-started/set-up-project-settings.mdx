## What To Configure First

A new project usually needs four things before it is ready for production:

1. Brand details and sender identity.
2. Domain authentication and DNS verification.
3. Team access and roles.
4. Integrations for wallet, contract, or event data.

## Launch Checklist

- Verify at least one sender.
- Authenticate the sending domain.
- Confirm branding matches the live product.
- Add the integrations needed for your first workflow.
- Run at least one internal test before broader use.
