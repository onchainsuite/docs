---
title: "Triggers and Conditions"
description: "Use triggers and conditions to control automation entry and branching."
---

## Common Triggers

- Wallet connected
- Contract action completed
- User joined a segment
- Webhook event received
- Time-based reminder reached

## Common Conditions

- User is in the correct segment
- User has not already received the message
- Wallet meets a contract or value threshold
- Required channel is available
